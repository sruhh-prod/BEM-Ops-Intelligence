{# ── Intro card ── #}

Pascal V1 answers your five most important daily ops questions — at-risk arrivals, access codes, STR compliance, data quality gaps, and your Thursday briefing. Run Pascal from the terminal while the full conversational interface is in development.

cd app && python3 main.py pascal
{# ── Commands ── #}

V1 Commands

At-Risk Arrivals
Which guests checking in soon have unresolved issues?
MVP
python3 main.py pascal arrivals
Access Code Check
Do all arriving guests have a door code set?
MVP
python3 main.py pascal access
STR Compliance
Which properties have expired or expiring STR licenses?
MVP
python3 main.py pascal compliance
Data Quality
Which properties are missing WiFi, door codes, or access instructions?
MVP
python3 main.py pascal gaps
Thursday Briefing
What do I need to cover in this week's ops call?
MVP
python3 main.py pascal thursday
